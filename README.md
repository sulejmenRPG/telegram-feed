# Telegram Feed

Модифицированный веб-клиент Telegram с функцией **новостной ленты (Feed)**.

## Что это?

Это форк [Telegram Web A](https://github.com/AzizDjan/telegram-tt) с добавленной функцией агрегированной ленты сообщений из выбранных каналов — по аналогии с новостной лентой в социальных сетях.

## Особенности

- **Feed-лента** — просматривай посты из нескольких каналов в едином потоке
- **Полный функционал Telegram Web** — чаты, звонки, стикеры, и всё остальное
- **PWA** — устанавливается как приложение на десктоп и мобильные устройства
- **Быстрый и лёгкий** — основан на кастомном фреймворке Teact

## Технологии

- **Teact** — React-подобный фреймворк с нуля
- **GramJS** — MTProto клиент для работы с Telegram API
- **WebAssembly** — для криптографии и обработки медиа
- **Web Workers** — многопоточность в браузере

## Локальная установка

```sh
cp .env.example .env
npm i
```

Получи API ID и API Hash на [my.telegram.org](https://my.telegram.org) и заполни `.env` файл.

## Запуск

```sh
npm run dev
```
